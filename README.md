# travel-booking-webpage

A modern, responsive travel booking landing page built with React, Vite, Tailwind CSS, and Framer Motion.

## Features
- **Top Flight Deals**: View available flight tickets with airline details and pricing.
- **Popular Destinations**: Explore top-rated travel spots.
- **Holiday Packages**: Specialized trip packages.
- **Luxury Cruises**: Discover ocean voyages.
- **Integrated Search**: Search for trips by origin and destination.
- **Responsive Design**: Fully mobile-friendly UI.

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Start the development server: `npm run dev`.
